# **App Name**: BlinkChat

## Core Features:

- Authentication: User authentication via email or social login (Google/Facebook).
- Profile Management: Profile creation and management (avatar, bio, interests).
- Interest-Based Chat: Random chat matching based on user-defined interests.
- Real-Time Chat: Real-time messaging between matched users.
- Follow/Unfollow Users: Follow/unfollow functionality for building connections.
- Block/Report Users: Blocking/reporting mechanism for handling unwanted interactions.
- Blink AI Assistant: AI tool that provides suggestions and assistance to users in chat.

## Style Guidelines:

- Primary color: Light purple (#A093FF) to convey a sense of modernity, intelligence, and interactivity.
- Background color: Very light purple (#F0F0FF) for a clean, translucent aesthetic in light mode.
- Accent color: Pink (#FFA3C9) used sparingly on highlights, message bubbles, and CTAs, contrasting with the primary for focus.
- Font pairing: 'Poppins' (sans-serif) for headlines, and 'PT Sans' (sans-serif) for body text, providing a balance of modern aesthetics and readability.
- Use minimalist line icons with subtle gradient overlays in pink and purple to maintain a consistent aesthetic.
- Employ translucent card components with frosted glass effect to create depth and a modern look.
- Incorporate subtle, interactive micro-animations for button presses and transitions to improve UX.
